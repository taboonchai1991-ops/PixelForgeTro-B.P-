✨ คำแนะนำ: อัปเกรดที่ควรทำตอนนี้
1. 🎯 Undo/Redo (Ctrl+Z, Ctrl+Y)

เป็นฟีเจอร์พื้นฐานที่ขาดไม่ได้ จะทำให้โปรแกรมดูเป็นมืออาชีพมากขึ้น

วิธีการ:

    เก็บประวัติการปรับแต่งใน Array

    แต่ละครั้งที่ปรับแต่ง → push state ลง history

    Ctrl+Z → pop กลับไป state ก่อนหน้า

    Ctrl+Y → redo ไปข้างหน้า

2. 🖼️ Crop / ตัดภาพ

เพิ่มความสามารถในการตัดภาพตามขนาดที่ต้องการ

วิธีการ:

    ใช้ Canvas + Mouse Event

    วาดกรอบสี่เหลี่ยมบนภาพ

    ลากมุมเพื่อปรับขนาด

    กดปุ่ม "ตัด" เพื่อใช้ภาพเฉพาะส่วน

3. 📝 Text Overlay

ใส่ข้อความบนภาพ เหมือนโปรแกรมแต่งรูปทั่วไป

วิธีการ:

    เพิ่มปุ่ม "เพิ่มข้อความ"

    เปิด Input ให้พิมพ์ข้อความ

    เลือกสี, ขนาด, ฟอนต์

    วาดข้อความบน Canvas ที่ตำแหน่งที่เลือก

4. 💡 AI Suggest (แนะนำการปรับแต่ง)

ใช้ AI วิเคราะห์ภาพแล้วแนะนำการปรับแต่งที่เหมาะสม

วิธีการ:

    วิเคราะห์ฮิสโตแกรมของภาพ

    แนะนำ: "ภาพนี้ควรเพิ่มความสว่าง 10%"

    หรือใช้ AI ตรวจจับว่าภาพคืออะไร (คน, ทิวทัศน์, อาหาร) แล้วแนะนำฟิลเตอร์ที่เหมาะสม

5. 🌓 Dark Mode / Light Mode

เพิ่มปุ่มสลับธีม ให้ผู้ใช้เลือกตามสบายตา

วิธีการ:

    ใช้ CSS Variables

    สร้างธีม Light และ Dark

    เปลี่ยน class บน body
